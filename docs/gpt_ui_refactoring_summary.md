# GPT UI 리팩토링 요약

## 📋 개요

현재 PE Agent의 UI를 ChatGPT와 유사한 3단 컬럼 레이아웃으로 리팩토링하기 위한 분석 및 구현 가이드입니다.

---

## 🎯 핵심 변경사항

### 현재 구조 → GPT 스타일 구조

| 항목 | 현재 | 변경 후 |
|------|------|---------|
| **레이아웃** | 상단 헤더 + 중앙 콘텐츠 (2단) | 왼쪽 사이드바 + 중앙 콘텐츠 + 오른쪽 사이드바 (3단) |
| **네비게이션** | 상단 수평 메뉴 | 왼쪽 세로 사이드바 |
| **다크 테마** | 지원 (기본) | GPT 스타일 색상 팔레트 적용 |
| **채팅 히스토리** | 없음 | 왼쪽 사이드바에 표시 |
| **보조 정보** | 없음 | 오른쪽 사이드바 (선택적) |

---

## 📐 레이아웃 구조

```
┌──────────┬──────────────────────┬──────────┐
│  Left    │   Main Content       │  Right   │
│ Sidebar  │   (최대 1200px)      │ Sidebar  │
│  288px   │                      │  320px   │
└──────────┴──────────────────────┴──────────┘
```

### 왼쪽 사이드바 구성
1. **로고 영역**: PE Agent 브랜딩
2. **새 채팅 버튼**: 새 대화 시작
3. **채팅 검색**: 히스토리 검색
4. **네비게이션 메뉴**: Chat, Search, Parsing, Retrieval Test
5. **채팅 히스토리**: 최근 대화 목록

### 중앙 콘텐츠 영역
- 페이지별 콘텐츠 표시
- 최대 너비 제한 (1200px) + 중앙 정렬
- 넓은 패딩으로 가독성 향상

### 오른쪽 사이드바 (선택적)
- AI 생각 과정 표시
- 참조 문서 목록
- 작업 진행 상태

## 🎨 디자인 시스템

### 색상 팔레트 (다크 테마)

```css
--color-bg-canvas: #000000      /* 전체 배경 */
--color-bg-base: #161618        /* 메인 배경 */
--color-bg-card: #1f1f23       /* 사이드바/카드 */
--color-text-primary: #f6f6f7  /* 주요 텍스트 */
--color-text-secondary: #b2b5b7 /* 보조 텍스트 */
```

### 타이포그래피
- 폰트: Inter, system-ui 계열
- 크기: 12px (xs), 14px (sm), 16px (base), 18px (lg)
- 가독성: line-height 1.5~1.7

### 간격
- 패딩: 12px, 16px, 24px 단위
- 아이템 간격: 4px, 8px, 12px

## 💡 구현 시 고려사항

### 1. 기존 코드 호환성
- 기존 페이지 컴포넌트는 그대로 사용 가능
- Layout 컴포넌트만 교체하면 됨

### 2. 상태 관리
- 채팅 히스토리는 로컬 스토리지 또는 백엔드 API 사용
- Context API 또는 Zustand 고려

### 3. 성능 최적화
- 가상 스크롤 (히스토리 목록)
- 메모이제이션 (메뉴 아이템)
- 코드 스플리팅 (페이지별)

### 4. 접근성
- 키보드 네비게이션 지원
- ARIA 레이블 추가
- 포커스 관리
