### 유사질문 포함 분석

- **bge_m3**: 전 구간 1위 유지. hit@1 48.42%, hit@10 71.84%
- **qwen3_emb_4b**: hit@10에서 bge_m3에 근접 (70.25% vs 71.84%). 대규모 후보군에서의 recall은 유사
- **jina_v5**: 전 구간 최하위. hit@10에서도 52.22%에 그침

### 원본 vs 유사질문 성능 변화

| model | hit@1 (원본→유사포함) | hit@5 (원본→유사포함) | hit@10 (원본→유사포함) |
|---|---|---|---|
| bge_m3 | 82.28 → 48.42 (-33.86) | 84.81 → 65.19 (-19.62) | 86.08 → 71.84 (-14.24) |
| jina_v5 | 10.13 → 16.14 (+6.01) | 89.87 → 45.25 (-44.62) | 91.14 → 52.22 (-38.92) |
| qwen3_emb_4b | 67.09 → 36.39 (-30.70) | 84.81 → 59.49 (-25.32) | 86.08 → 70.25 (-15.83) |

- 유사질문은 비공식 용어, 줄임말, 구어체 표현이 많아 전반적으로 성능 하락
- **bge_m3의 하락폭이 가장 적음** — 다양한 표현에 대한 로버스트니스가 상대적으로 우수
- jina_v5는 원본에서 hit@5/10이 높았으나 유사질문에서는 크게 하락 — 특정 표현에 과적합된 패턴

## 종합 결론

| 평가 항목 | bge_m3 | jina_v5 | qwen3_emb_4b |
|-----------|--------|---------|--------------|
| Top-1 정밀도 | 1위 | 3위 | 2위 |
| Top-10 Recall | 1위 (유사질문) / 2위 (원본) | 1위 (원본) / 3위 (유사질문) | 2위 |
| 표현 로버스트니스 | 1위 | 3위 | 2위 |
| 인덱스 크기 | 7.5GB | 3.8GB | 24.3GB |
| 임베딩 차원 | 768 | 1024 | 2560 |

### 권장 사항

1. **bge_m3를 기본 임베딩 모델로 유지** — 정밀도, 로버스트니스, 인덱스 효율 모두 최적
2. jina_v5는 top-1 정밀도가 너무 낮아 실용성 부족
3. qwen3_emb_4b는 인덱스 크기(24.3GB) 대비 bge_m3보다 나은 점이 없음
4. 향후 검색 품질 개선은 임베딩 모델 변경보다 **챕터 그룹핑 검색** 등 검색 전략 개선에 집중 권장
